My Journey to UTY

Seperti banyak orang, saya tidak berhasil lolos seleksi masuk universitas pilihan saya melalui jalur SNBP. Saya mencoba lagi melalui jalur SNBT, namun hasilnya tetap sama. Karena sejak dulu saya ingin berkuliah di Yogyakarta, saya mulai mencari universitas yang memiliki akreditasi baik untuk jurusan Informatika.

Dalam pencarian tersebut, saya mendapatkan informasi tentang Universitas Teknologi Yogyakarta (UTY) dari ayah saya, yang membantu saya saat survei beberapa universitas di Yogyakarta. Akhirnya, saya memutuskan untuk mendaftar di UTY dan berharap dapat menggapai impian saya untuk meraih kesuksesan di bidang IT.
